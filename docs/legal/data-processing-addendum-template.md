# Data Processing Addendum Template

## Roles
Identify whether the company is controller, processor, service provider, or business associate for each data category.

## Processing Details
Describe subject matter, duration, nature, purpose, data categories, and data subjects.

## Security
List access controls, encryption, audit logging, vendor review, incident response, and deletion/return process.

## Subprocessors
List subprocessors, notice process, objection process, and locations where data is processed.

## Assistance
Describe support for data subject requests, DPIAs, audits, breach notices, and regulator inquiries.

## Deletion or Return
Define when customer data is deleted, returned, anonymized, or retained for legal obligations.
